01 · CONTEXT INSPECTOR

From fingerprint to scalpel.

The 200-cell grid is a live readout, not a static viz. Expand any category to see its constituent items — every tool, sub-agent, memory file, skill — each with its own token cost. Toggle them off and the dots, headline, per-cat totals, and savings pill all re-tween in <400ms. Locked rows (system prompt, message history) wear a small lock glyph; in-use items pulse an in-use tag so you know not to disable an active tool. Bulk actions live inside each open category.

02 · COORDINATOR PROMINENCE

Anchor the current thread.

Coordinator gets a dedicated section header and a left accent bar — it's where the user is, not just another item in a list. Diamond glyph + accent color makes it unmistakable across modes (sidebar / dock / sheet).

03 · DISTILLED CARDS

Two lines, not seven.

Each agent fits in a clean grid: provider glyph · name + model · current task · live status LED. A 1px top-edge strip on the card itself shows context fill — warming to amber past 70%, coral past 90%. Timestamps, mode flags, diff stats, and action buttons hide behind hover — surfacing only when wanted.

04 · STATUS AS A SIGNAL

Lime means living.

One accent (electric lime) is reserved exclusively for active state — pulsing LED on running agents, glow on the active thread, beat in the footer. Errors warm to coral, done agents go cool. The eye knows where to look without reading.
